‚ö° Batch Processing

POST /api/batch/process 200 400
Process all pending transactions in a batch using zero-knowledge proofs. This updates account balances and generates a new Merkle root.
Response:
{ "success": true, "message": "Batch processed successfully", "processed_count": 3, "new_root": "0x2bb9b82c29f5c7d85b08f9c8908d648c454e317fbb43b51beb9b632ce3049c1", "receipt_saved": true }

Verify a zero-knowledge proof receipt to ensure the batch processing was valid.
